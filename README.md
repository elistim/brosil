# 🦊 Brosil — трекер привычек

Простой веб-сервис для отслеживания:
- 🚭 дней без курения
- 🍺 дней без алкоголя
- 📓 ведения дневника

---

## 🚀 Возможности

- 📅 Календарь с отметками по дням
- 🔢 Счётчик непрерывной серии (streak)
- 🎯 Выбор любого дня и редактирование
- 💾 Сохранение данных в `data.json`
- 🌐 Доступ с любых устройств
- 🦊 Интерактивный лисёнок

---

## 🏗 Стек

- FastAPI
- HTML / CSS / JS
- nginx + uvicorn

---

## ▶️ Запуск

uvicorn app:app --host 127.0.0.1 --port 8090

## 🦊 Поведение лиса

- бегает за курсором
- убегает
- засыпает у края
- просыпается по клику


# Недостатки
- Только для одного пользователя
- Видят все, кто имеет доступ
- Нет возможности отключить лиса
